# Kodemirror

**CodeMirror 6, ported to Kotlin Multiplatform and Compose.**

Kodemirror brings the power of CodeMirror's editor framework to Kotlin
and Compose Multiplatform, providing a rich code editing experience on
JVM and WebAssembly targets.
